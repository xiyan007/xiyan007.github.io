    框架

    框架是为了解决一类问题而开发出来的产品，库是功能代码模块的集合，供开发者调用。

    Ant Design

    antd 是基于 Ant Design 设计体系的 React UI 组件库，主要用于研发企业级中后台产品。

    Ant Design Pro

    Ant Design Pro 是基于 Ant Design 和 umi 的封装的一整套企业级中后台产品的前端/设计解决方案。

    Umi

    Umi 是蚂蚁金服的底层前端框架。

    umi 具备的功能：

    • 可扩展，Umi 实现了完整的生命周期，并使其插件化，Umi 内部功能也全由插件完成。
    • 开箱即用，Umi 内置了路由、构建、部署、测试等，仅需一个依赖即可上手开发。
    • 完备路由，同时支持配置式路由和约定式路由，比如动态路由、嵌套路由、权限路由等等。

    什么时候不用 umi？

    • 需要支持 IE 8 或更低版本的浏览器
    • 需要支持 React 16.8.0 以下的 React
    • 需要跑在 Node 10 以下的环境中
    • 有很强的 webpack 自定义需求和主观意愿
    • 需要选择不同的路由方案

    DvaJS

    dva 首先是一个基于 redux 和 redux-saga 的数据流方案，然后为了简化开发体验，dva 还额外内置了 react-router 和 fetch，所以也可以理解为一个轻量级的应用框架。

    Quasar

    Quasar 是一个基于麻省理工学院的授权开源Vue.js的框架，它可以快速创建一个桌面网站，一个桌面应用程序，一个移动应用程序。

    Cordova

    Cordova 是一个开源的移动端应用框架，允许使用标准的web技术，html5，css3 和js 进行跨平台开发

    Element

    Element 是一个基于Vue 2.0的组件库，面向开发人员、设计人员和产品经理

    NodeJS

    允许 JS 代码在服务端运行。

    Express

    Express 是 Node.js 社区广泛使用的框架，简单且扩展性强，非常适合做个人项目。但框架本身缺少约定，标准的 MVC 模型会有各种千奇百怪的写法。

    EggJS

    Egg 按照约定进行开发，奉行『约定优于配置』，团队协作成本低。

    • 提供基于 Egg 定制上层框架的能力
    • 高度可扩展的插件机制
    • 内置多进程管理
    • 基于 Koa 开发，性能优异
    • 框架稳定，测试覆盖率高
    • 渐进式开发
