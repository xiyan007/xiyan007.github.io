    CSS 绘制三角形

    设置元素 width 和 height 为 0，当每个边框设置相同宽度，不同颜色时，会呈现4个不同颜色的三角形，可以设置其他边框的宽度和背景色，只显示一个三角形。

    绘制一个箭头向下的三角形 

    1
2
3
4
5
6
7
    div {
  width: 0;
  height: 0;
  border-top: 20px solid #000;
  border-left: 20px solid transparent;
  border-right: 20px solid transparent;
}

  }

  <div class="child"></div>

    有以下几种方案：

    • flex
    • position + transform
    • position + margin: auto
    • grid
    • table-cell

    flex

     1
 2
 3
 4

 6

 9

    .parent {
  /* 父元素设置为弹性容器, 弹性项目默认在水平轴上排列 */
  display: flex;
  /* 设置弹性项目在交叉轴（垂直）方向上的对齐方式，居中 */
  align-items: center; 
  /* 
    分配弹性项目之间的距离，
    center: 每行第一个元素到行首的距离将与每行最后一个元素到行尾的距离相同 
  */
  justify-content: center;
}

.child {
  position: absolute;
  /* 设置子元素坐上角位于父元素中心 */
  top: 50%;
  left: 50%;
  /* 使子元素向左上角方向偏移自身宽高一半的值，使父子元素中心点重合 */
  transform: translate(-50%, -50%);
}
