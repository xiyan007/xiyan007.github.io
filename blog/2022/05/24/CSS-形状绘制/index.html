三角形绘制原理

绘制原理：设置元素宽高均为 0，通过设置边框 border 来绘制三角形。

step1: css border 绘制宽为 100px 的正方形，每个边框设置不同颜色

step2: 保留一个边框颜色，其他三个边框设置透明

这样可以绘制一个箭头向下的三角形，但是元素实际占用尺寸比显示的要多。

step3: 分别设置边框大小，使绘制的图形占用尺寸最小

分别设置左、上、右边框大小为 50px, 再设置左右边框颜色透明。
